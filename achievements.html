  • Tiash Ghosh, Ratul Kishore Saha, Amit Achuthanunni, Shomesh Kumar Singha (Electrical Engineering Department, Future Institute of Engineering and Management) was selected in top six project for AERO INDIA 2019 (20-24th February Bengaluru) and awarded with commendation certificate. Project Name: A small- scale Quardcopter for real time detection, classification and localization. The project aims to evolve a system along with suitable software for automatic detection,classification and localization of ground activity from a quadcopter, an unmanned aerial vehicle system (UAV), equipped with a miniature camera. The quadcopter processing unit trained with control algorithms and the images captured by quadcopter camera, will be analyzed with a predesigned deep neural network. Once detected, the object will be classified and movement of object of interest will be locked for visual tracking in the target scene, if needed. The UAV system will run a robust deep neural algorithm for detection purpose and the activity classification is accomplished by hybrid correlation filter.
